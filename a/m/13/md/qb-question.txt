::1984::[html]<p>1984 從小說到現實</p>{
	= 1940 年代， [[A]] 預見未來科技的發展， 有可能讓政府 (以及書中的統治者 「老大哥」) 隨時監控兼洗腦民眾， 於是寫下 「一九八四」 這部反烏托邦 (dystopia) 的科幻小說。 -> George Orwell
	= 真實世界的 1984 年， 壟斷電腦市場的是 [[B]] 公司。 蘋果電腦在超級盃足賽當中推出一段 kuso 「一九八四」 小說的影片， 宣稱要將大眾從 [[B]] 老大哥的掌控當中解放出來。 -> IBM
	= 同年， [[C]] 創立自由軟體基金會並啟動 GNU 計畫， 主張軟體使用者應該享有使用/研究/分享/修改等四大自由。 -> Richard M. Stallman
	= 2009 年， [[D]] 公司從遠端把手伸進其客戶的 kindle 電子書裡， 刪除兩本書， 完全實現 (甚至超越) 了 (A) 的預言。 被刪的書， 正好就是 (A) 的 「一九八四」 跟 「動物農莊」。 -> Amazon
	= 2013 年， [[E]] 爆料美國國安局 NSA 濫權監聽， 包含在 Windows 植後門。 各界驚覺 (C) 主張軟體原始碼必須公開的重要性： 沒有原始碼， 根本就不需要談資訊安全了。 外洩的 NSA 簡報當中指出： 「誰會想到 1984 書中的老大哥竟然是賈伯斯， 而愛用者正是書中被控制的行屍走肉?」 -> Edward Snowden
	= 同年， 我國行政院資通會報技術服務中心推出一份顛倒黑白的「行動裝置資通安全注意事項」 把蘋果電腦的遠端監控技術吹捧成保護用戶的資安設計。 其中的 [[F]] 技術讓集權的中央伺服器可以從遠端操控與監聽公教人員手機， 比 (D) 公司的 kindle 電子書的搖控效果更加徹底。 -> Mobile Device Management
	= -> Aaron Swartz
	= -> 指紋解鎖
	= -> Microsoft
}

::硬碟分割概念::[html]<p>硬碟分割的相關知識</p>{
	= 一顆硬碟裡， 儲存 boot loader、 掌管開機的那一塊空間稱為 [[A]]。 此處無法以檔案總管拷貝。 -> Master Boot Record
	= 一顆硬碟除去 (A) 之外， 可以分成四塊 [[B]]。 在 linux 底下， 其代號永遠是 1 或 2 或 3 或 4。 -> primary partitions
	= 如果四個 (B) 不夠用， 還可以把其中一塊變成 [[C]]。 它其實只是一個空殼。 -> extended partition
	= (C) 裡面可以切成很多塊 [[D]]， 每塊都可以格式化 (format) 成為一個獨立的檔案系統。 不論硬碟上已有多少個 (B)， 每個 (D) 的代號永遠都是 5 以後的數字。 -> logical partition
	= -> grub 
	= -> directory
}
